# 🌺HTML 介绍

Web 前端三大核心技术

  • HTML：负责网页的架构
  • CSS：负责网页的样式、美化
  • JS：负责网页的行为

什么是 HTML

  • HTML 是用来描述网页的一种语言

HTML 标签

  • 单标签

  • 双标签

标签属性

  • 属性格式：属性名 = "属性值"

# 🌺HTML 常用标签

# 🌱标题标签

HTML 标题是通过 <h1> - <h6> 等标签进行定义

# 🌱段落标签

HTML 段落是通过 <p> 标签进行定义的

# 🌱超链接标签

超链接是通过 <a> 标签进行定义的

属性

  • href：跳转地址
  • target：新窗口打开

# 🌱图片标签

HTML 图片是通过 <img> 标签进行定义的

属性

  • src：图片路径
  • title：光标悬停显示文字
  • alt：图片未加载时显示文字
  • width：图片宽度
  • height：图片高度

# 🌱换行和空格

换行：<br/>

空格：&nbsp;

# 🌱布局标签

页面布局常用 <div> </div> 和 <span> </span>

# 🌱列表标签

列表标签常用 li 元素（分为：有序和无序）

# 🌱表单标签

页面提交输入信息需要使用表单标签 <form>
