# ⛵️Docker 数据卷

# 🐬概念

  • 数据卷是宿主机的一个目录或文件
  • 当容器目录和数据卷目录绑定后，对方的修改会立即同步
  • 一个数据卷可以被多个容器同时挂载
  • 一个容器也可以被挂载多个数据卷
  • 数据卷和容器之间是多对多关系

# 🐬作用

  • 容器数据持久化
  • 外部机器和容器间接通信
  • 容器之间数据交换

# 🐬配置

创建并启动容器时，使用 -v 参数设置容器卷

  1. 目录必须是绝对路径
  2. 如果目录不存在，则会自动创建
  3. 可以挂载多个数据卷
